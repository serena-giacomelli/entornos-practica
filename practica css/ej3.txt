Ejercicio 3
Analizar el siguiente código señalando declaraciones y aplicaciones de reglas, y su efecto.
p.quitar {
color: red;
}
*.desarrollo {
font-size: 8px;
}
.importante {
font-size: 20px;
}
<p class="desarrollo">
En este primer párrafo trataremos lo siguiente:
<br />xxxxxxxxxxxxxxxxxxxxxxxxx
</p>
<p class="quitar">
Este párrafo debe ser quitado de la obra…
<br />xxxxxxxxxxxxxxxxxxxxxxxxx
</p>
<p >
En este otro párrafo trataremos otro tema:<br />
xxxxxxxxxxxxxxxxxxxxxxxxx
</p>
<p class="importante">
Y este es el párrafo más importante de la obra…
<br />xxxxxxxxxxxxxxxxxxxxxxxxx
</ p>
<h1 class="quitar">Este encabezado también debe ser quitado de la obra</h1>
<p class="quitar importante">Se pueden aplicar varias clases a la vez</p>


1. Declaraciones de reglas CSS:

- p.quitar { color: red; }: Aplica a los elementos <p> con clase "quitar". El texto será rojo.
- *.desarrollo { font-size: 8px; }: Aplica a cualquier elemento con clase "desarrollo". El texto será de 8px.
- .importante { font-size: 20px; }: Aplica a cualquier elemento con clase "importante". El texto será de 20px.

2. Aplicaciones y efecto:

- <p class="desarrollo">...: Se aplica la regla de *.desarrollo. El texto será de 8px.
- <p class="quitar">...: Se aplica la regla de p.quitar. El texto será rojo.
- <p> (sin clase): No se aplica ninguna regla, mantiene estilos por defecto.
- <p class="importante">...: Se aplica la regla de .importante. El texto será de 20px.
- <h1 class="quitar">...: No se aplica ninguna regla, ya que p.quitar solo afecta a <p>, y no hay regla para h1.quitar.
- <p class="quitar importante">...: Se aplican ambas reglas: el texto será rojo y de 20px.

3. Observaciones:

- El selector *.desarrollo es equivalente a .desarrollo porque la clase es única.
- Se pueden aplicar varias clases a la vez, combinando los estilos de cada una.
- Las reglas solo afectan a los elementos que coinciden exactamente con el selector.
